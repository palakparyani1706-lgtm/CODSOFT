# 📝 To-Do List Application (Python)

## 📌 Overview
This is a simple yet effective **To-Do List Application** built using Python.  
It helps users manage their daily tasks by allowing them to add, update, delete, and track tasks.

The project includes both CLI and GUI versions.

---

## ✨ Features
- ➕ Add new tasks  
- ✏️ Update existing tasks  
- ❌ Delete tasks  
- 📋 View all tasks  
- 💾 Persistent storage using JSON file  
- 🖥️ CLI + GUI support  

---

## 📁 Project Structure
todo.py → Command-line application logic  
gui.py → Graphical User Interface (GUI) version  
task.json → Stores task data in JSON format  

---

## ⚙️ Requirements
- Python 3.x  
- Tkinter (pre-installed with Python)

---

## ▶️ How to Run

### CLI Version
python todo.py

### GUI Version
python gui.py

---

## 💾 Data Storage
All tasks are stored in:
task.json

This file updates automatically when tasks are modified.
