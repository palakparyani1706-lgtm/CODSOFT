# 🔐 Password Generator (Python + GUI)

## 📌 Overview
This is a simple **Password Generator application** built using Python.  
It generates strong and random passwords based on user-defined length and complexity.

The project includes both **Command-Line (Python)** and **Graphical User Interface (GUI)** versions.

---

## ✨ Features
- 🔢 User can specify password length  
- 🔐 Generates strong and random passwords  
- 🧠 Uses combination of letters, numbers, and symbols  
- 🖥️ GUI-based simple interface  
- ⚡ Fast and easy to use  

---

## 📁 Project Structure
password_generator.py → Python logic for generating passwords  
password_generator_gui.py → Graphical User Interface (GUI version)  

---

## ⚙️ Requirements
- Python 3.x  
- Tkinter (usually pre-installed with Python)

---

## ▶️ How to Run

### 🔹 Run Python Version (CLI)
python password_generator.py

---

### 🔹 Run GUI Version
password_generator_gui.py

---

## 🔐 How It Works
1. User enters desired password length  
2. Program generates a random password  
3. Password is displayed on screen  
4. User can copy and use it  

---

## 🎯 Purpose of the Project
This project was created to understand:
- Random module in Python  
- String manipulation  
- User input handling  
- GUI development using Tkinter
