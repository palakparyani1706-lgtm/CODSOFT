# 📒 Contact Book Application (Python + GUI)

## 📌 Overview
This is a simple **Contact Book Application** built using Python.  
It helps users store, manage, and organize their contact information efficiently.

The project includes both **Command-Line (Python)** and **Graphical User Interface (GUI)** versions.

---

## ✨ Features
- ➕ **Add new contacts** (Name, Phone, Email, Address)  
- 📋 **View contact list**  
- 🔍 **Search contacts** by name or phone number  
- ✏️ **Update contact details**  
- ❌ **Delete contacts**  
- 💾 **Data stored in JSON file**  
- 🖥️ **User-friendly GUI interface**  

---

## 📁 Project Structure
contact_manager.py → Command-line contact management logic  
contact_manager_gui.py → Graphical User Interface (GUI version)  
contacts.json → Stores all contact data  

---

## ⚙️ Requirements
- Python 3.x  
- Tkinter (usually pre-installed with Python)

---

## ▶️ How to Run

### 🔹 Run CLI Version
python contact_manager.py

---

### 🔹 Run GUI Version
python contact_manager_gui.py

---

## 💾 Data Storage
All contacts are stored in a local JSON file:

contacts.json

This file automatically updates whenever contacts are added, edited, or deleted.

---

## 🎯 Purpose of the Project
This project was created to understand:
- File handling using JSON  
- CRUD operations (Create, Read, Update, Delete)  
- Search functionality  
- GUI development using Tkinter  
- Real-world application design
